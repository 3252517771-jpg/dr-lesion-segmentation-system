## 三、模块边界与职责划分

### 3.1 架构分层

```
┌─────────────────────────────────────────────┐
│  Pages（页面层）                              │
│  职责：路由节点，组合 components 组装完整页面  │
│  依赖：components/*, hooks/*, api/*          │
│  注意：页面不写业务逻辑，只做组合编排          │
├─────────────────────────────────────────────┤
│  Components（UI 层）                          │
│  职责：通用 UI 组件 + 诊断组件 + 3D + 图表    │
│  依赖：types/*, styles/*                     │
│  注意：纯展示，通过 props 驱动                │
├─────────────────────────────────────────────┤
│  Hooks / API（数据层）                        │
│  职责：TanStack Query hooks + axios 请求     │
│  依赖：types/*, client.ts                    │
│  注意：不与 UI 直接耦合                      │
├─────────────────────────────────────────────┤
│  Types / Styles（基础层）                     │
│  职责：类型定义、全局样式                     │
│  依赖：无                                    │
│  注意：不允许依赖上层模块                      │
└─────────────────────────────────────────────┘
```

### 3.2 依赖规则

```
Pages → Components → 基础层
Pages → Hooks/API → 基础层

禁止：Components 依赖 Pages
禁止：API 依赖 Components
禁止：基础层 依赖任何上层
```

## 六、核心接口协议

### 6.1 API 请求封装

```typescript
// src/api/client.ts
import axios from "axios";

const client = axios.create({
  baseURL: "/api",
  timeout: 60000,       // 推理可能较慢，60s 超时
});

// 响应拦截：统一错误处理
client.interceptors.response.use(
  (res) => res,
  (error) => {
    const message = error.response?.data?.error?.message || "网络错误";
    // Ant Design message.error(message)
    return Promise.reject(error);
  }
);

export default client;
```

## 七、Three.js 3D 组件设计

### 7.1 LesionSphere 组件

```typescript
// src/components/three/LesionSphere.tsx
// 精简版，≤200 行

interface LesionSphereProps {
  lesionData: {
    type: string;       // "HE" | "EX" | "MA" | "SE"
    positions: Array<{x: number, y: number}>;   // 2D 坐标
    count: number;
  }[];
}
```

**实现逻辑：**
```
1. 接收 4 类病灶的 2D 掩码坐标
2. 将 (x, y) 映射到球面 (lat, lon)
   → 3D 坐标 (sin(lat)*cos(lon), sin(lon), cos(lat)*cos(lon))
3. 渲染：
   - 半透明灰色球体（SphereGeometry）
   - 每个病灶用对应颜色的小方块（BoxGeometry）定位在球面
   - 方块大小 = 病灶面积占比的映射值
4. OrbitControls 提供拖拽旋转 + 滚轮缩放
5. 右上角固定显示 LesionLegend（类别颜色图例）
6. 点击病灶方块可显示 tooltip（该类型名称 + 数量 + 面积占比）
```

### 7.2 渲染时机

- 仅在诊断详情页（`/diagnose/:id`）中渲染
- 默认隐藏在 Tab "3D 分布" 中，切换到时才加载（避免页面卡顿）

## 十一、质量底线

| 检查项 | 标准 |
|--------|------|
| 无控制台报错 | `console.error` 和未捕获异常数 = 0 |
| 无 TypeScript 错误 | `tsc --noEmit` 通过 |
| 组件 Props 有类型 | 无 `any` 滥用 |
| 路由切换不白屏 | 所有路径可访问 |
| 图片加载有兜底 | 前端正确显示 404 占位图 |
| Three.js 懒加载 | 切换到 3D Tab 时才渲染，不在页面初始化时加载 |
